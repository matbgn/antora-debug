Introduction to Node

In order to run your program locally like Hello world, proceed as following:

  1. Create a new file with extension .js like hello.js

  2. Put the following code in it:

printHello("hello") // Invoke the printHello function below

function printHello(word) {
  setTimeout(function () {
    console.log('World!');
  }, 2000);
  console.log('Hello');
}

  1. Then type in your terminal the following command:

$ node hello.js

  1. You should see:

Hello
World! // 2 sec after
